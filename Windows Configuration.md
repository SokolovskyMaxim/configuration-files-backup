# OS-W-CNF

>[!warning] ANOTHER ED IS REQUIRED

## Quick Start 

### Install

- Shift + F10
- `OOBE\BYPASSNRO`
- `ipconfig /release`

### Initial preparation

```
slmgr/ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
```
```
slmgr /skms kms.digiboy.ir
```
```
slmgr /ato
```
```
iwr -useb https://git.io/debloat|iex
```
